# BoutiqueComplete1

A full-stack Shop Management System using **MongoDB** and **Flask**. 

**Features:**
- **Products**: CRUD, Search, Filters.
- **Orders**: Management using Embedding & Linking patterns.
- **Analytics**: Aggregation pipelines for sales & stock.
- **Tech**: Python, PyMongo, HTML/CSS/JS.

**How to Run:**
```bash
pip install -r requirements.txt
python db_init.py
python app.py
```
Open **http://localhost:5000**
